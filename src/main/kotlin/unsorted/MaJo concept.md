# Main Journal (MaJo)

MaJo - это приложение для меня? Или для кого-то еще?
Наверное так-как это мой пет-проект - то это приложение **ДЛЯ МЕНЯ**.

## Функционал

Что должно уметь приложение:
- Фиксировать элементарные действия
   - Занятия спортом (бег, спортзал)
   - Занятия образованием (программированием)
   - Занятия медитацией
- Фиксировать финансовые потоки
   - Расходы
   - Доходы
- 

## Сущности MaJo
- Запись
    - Категория (спорт/образование/питание)

## Более подробное описание сущностей

- Запись - метка, заметка, очерк, записка в общем сути отражающее какое-то ДЕЙСТВИЕ
