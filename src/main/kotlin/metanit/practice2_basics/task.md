### Задача: Анализ успеваемости студентов

`Данное задание должно покрыть все темы пройденные во второй главе`

Напиши программу на Kotlin, которая выполняет следующие действия:

1. **Запрос количества студентов**  
   Программа выводит сообщение: *«Введите количество студентов:»* и считывает целое число `n` с клавиатуры. Учти, что 
2. пользователь может ввести не число — в этом случае программа должна выдать сообщение об ошибке и завершиться (пока можно без повторного ввода).

2. **Создание массива для оценок**  
   Создай массив целых чисел размером `n`. В Kotlin для этого можно использовать `IntArray(n)`.

3. **Заполнение массива оценками**  
   Организуй цикл `for` от `0` до `n-1` (или используй диапазон `0 until n`). На каждой итерации запрашивай оценку для студента с номером `i+1` (например, *«Введите оценку для студента 1 (от 2 до 5):»*).  
   Проверь корректность ввода:
    - Считанное значение должно быть целым числом (обработай возможное исключение или проверку через `toIntOrNull`).
    - Оценка должна лежать в диапазоне от 2 до 5 включительно (используй `in 2..5`).  
      Если ввод некорректен — выведи сообщение и попроси ввести оценку ещё раз (можно организовать внутренний цикл `while`).

4. **Меню действий**  
   После заполнения массива программа должна выводить меню:
   ```
   Выберите действие:
   1. Вывести все оценки
   2. Вычислить среднюю оценку
   3. Найти максимальную оценку
   4. Найти минимальную оценку
   5. Посчитать количество отличников (оценка 5)
   6. Проверить наличие двоек
   7. Выйти
   ```
   Для выбора пункта используй `when`. Программа должна выполняться в цикле `while`, пока пользователь не выберет пункт 7.

5. **Реализация пунктов меню**
    - **Пункт 1**: выведи все оценки через пробел (можно в цикле `for`).
    - **Пункт 2**: вычисли сумму всех оценок (цикл `for`) и раздели на количество студентов. Результат должен быть вещественным числом (`Double`). Выведи с двумя знаками после запятой.
    - **Пункт 3**: найди максимум в массиве, используя цикл и сравнение (`if`).
    - **Пункт 4**: найди минимум аналогично.
    - **Пункт 5**: посчитай, сколько элементов массива равны 5 (цикл с условием `if`).
    - **Пункт 6**: проверь, есть ли в массиве хотя бы одна оценка 2. Для этого можно пройти по массиву и, если встретилась двойка, вывести *«Есть двоечники»* и завершить проверку. Если после цикла двойка не найдена — вывести *«Двоек нет»*.
    - **Пункт 7**: заверши программу.

6. **Дополнительные требования**
    - Используй **переменные** для хранения ввода, промежуточных результатов.
    - Обрати внимание на **типы данных**: `Int` для целых чисел, `Double` для среднего, `Boolean` для флагов.
    - Примени **операции над числами**: сложение, сравнение, деление, возможно `%` для проверки чётности (не обязательно).
    - Работа с **диапазонами** уже встречалась при проверке оценки (`2..5`) и может быть полезна при генерации индексов.
    - Массивы — это основа всей задачи, ты будешь обращаться к элементам по индексу.

### Подсказки

- Для чтения целого числа используй `readln().toIntOrNull()`. Если вернулся `null` — ввод некорректен.
- Для вывода вещественного числа с двумя знаками можно использовать `String.format("%.2f", average)` или `"%.2f".format(average)`.
- Внутри цикла заполнения массива удобно использовать индексы: `grades[i] = ...`
- Для повторения меню нужна переменная-флаг, например `var isRunning = true`, и цикл `while (isRunning)`. В пункте 7 меняй флаг на `false` или используй `break`.

- Используй `when` без аргумента для проверки нескольких условий в одном блоке (например, при валидации оценки).
